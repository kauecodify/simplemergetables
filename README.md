# simplemergetables

---

Funções da Interface

1) Carregar um arquivo CSV ou Excel

Você clica no botão “Carregar Arquivo”
→ Abre uma janela para escolher seu arquivo.
→ A aplicação lê o arquivo e mostra as colunas.

2) Mostrar todas as colunas da tabela

Depois que o arquivo é carregado:

A lista de colunas aparece na tela
Você pode selecionar múltiplas colunas (Shift / Ctrl / clique)

3) Mesclar (juntar) várias colunas em uma só

---

Você escolhe:

quais colunas quer juntar

o nome da coluna nova

o separador (ex.: espaço, traço, vírgula, etc.)

Exemplo:
Colunas: Nome, Sobrenome, Cidade
Separador: " - "
Nova coluna: DadosCompletos

Resultado:
nome - sobrenome - São Paulo

4) Salvar o resultado em um novo arquivo Excel

Depois de mesclar:

Você escolhe onde salvar
O arquivo final sai como .xlsx
A nova coluna aparece no final da tabela

---

Em resumo

A interface permite você:

- Carregar tabela
- Ver colunas
- Selecionar várias
- Mesclar em 1 coluna só
- Exportar pronto
